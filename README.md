# Bank Management System 💳

A simple Bank Management System built in Java using Object-Oriented Programming (OOP) concepts and File Handling.

## Features 
- User Registration
- User Login Authentication
- Admin Registration
- Client Management
- Balance Management
- User Data Storage using File Handling
- Update Existing User Data

## Technologies Used 
- Java
- OOP (Inheritance, Encapsulation)
- File Handling
- Collections Framework


## How It Works 
1. Register as Client or Admin
2. Login using username and password
3. Manage account balance
4. Store and update user data in text files

## OOP Concepts Used 
- Inheritance
- Method Overriding
- Constructors
- Encapsulation

## File Handling 
The project stores user data in:
```txt
users.txt
```

Format:
```txt
username,password,balance,role
```

Example:
```txt
ali,1234,5000,client
admin,admin123,0,admin
```
